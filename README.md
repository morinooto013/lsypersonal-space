# Velorah

A cinematic personal website with a fullscreen video background, glassmorphic UI, and smooth entrance animations.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Deployed on Vercel

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Deploy

Deployed automatically via Vercel on every push to `main`.
